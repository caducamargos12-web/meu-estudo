Como adicionar / remover alunos
Para adicionar ou remover um aluno, edite a variável ALUNOS no Railway, no formato usuario:senha separados por vírgula.

O botão Liberar dispositivos abaixo serve para quando um aluno trocou de aparelho ou limpou o navegador e ficou travado. Cada aluno pode usar até 2 dispositivos (ex: celular + PC). Liberar zera os aparelhos vinculados para ele cadastrar de novo.
